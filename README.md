# Proyecto: Moodboard Personal con CSS GRID

Este proyecto consiste en la creación de un moodboard personal funcional y responsive, siguiendo un modelo estructural específico proporcionado en clase.

---

## Autor
**Diego Ferreiro Couceiro** 

---

## Descripción de la Práctica
El objetivo es utilizar **CSS GRID** para la disposición de los elementos de un moodboard.El diseño refleja mi personalidad a través de colores, tipografías y sombras propias, manteniendo la estructura obligatoria.

**Características clave:**
* **Semántica:** Uso estricto de etiquetas HTML5 validadas por el W3C.
* **Interactividad:** Implementación de efectos *hover* para dar foco a los elementos.
* **Responsive:** Diseño adaptable mediante Media Queries para cualquier pantalla.
* **Código limpio:** Uso de variables CSS para colores y anidamiento de reglas.
---
## Tecnologías Utilizadas
* **HTML5:** Marcador semántico.
* **CSS3 (Grid Layout):** Estructura principal y diseño adaptativo.
* **Variables y Anidamiento CSS:** Para una mejor organización del código.
* **GitHub:** Para el control de versiones y entrega del proyecto.
